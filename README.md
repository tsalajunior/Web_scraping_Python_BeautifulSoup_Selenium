# Web_scraping_Python_BeautifulSoup_Selenium

cet algorithme collecte les données depuis un site de réservation de voyage 
la recherche a été affinée avec des critères tels que la destination de départ, la durée du séjour, etc.
à la fin on a des données exploitables sous forme de tableur Excel

les technologies utilisées sont les suivantes :
- Python
- Selenium
- BeautifulSoup
- Pandas
- Seaborn
- Matplotlib
- openpyxl
